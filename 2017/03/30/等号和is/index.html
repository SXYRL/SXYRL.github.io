关于 == 和 is

判断内容的时候使用==

#这里比较的是两个对象中的值，而不进行内存地址的检测，返回布尔值

>>>a = [1,2,3]    
>>>b = [1,2,3]
>>>a == b
True

判断是否指向同一个对象的时候用is

>>>a = [1,2,3]    
>>>b = [1,2,3]
>>> id(a)
45600072
>>> id(b)
45721032
>>> a is b
False

这里需要注意的是：

1,整数在程序中的使用非常广泛，Python为了优化速度，使用了小整数对象池，避免为整数频繁申请和销毁内存空间。
python对小整数的定义是[-5,257],也就是-5到256且包含256，这些整数对象是提前建立好的，在一个python的程序中，所有位于这个范围内的整数使用的都是同一个对象。
同理，单个字母也是这样的。
2,如果定义的变量都是普通字母，不包含其他特殊符号的，默认开启intern机制，共用对象，不会进行新建
例子：

>>> a = 100
>>> b = 100
>>> a is b
True

>>> a = 'hehe'
>>> b = 'hehe'
>>> a is b
True
